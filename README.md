## 🎯 Final Project Target: "Warung Digital" (Digital Grocery Store)
The ultimate goal of this learning path is to build a **Warung Digital Application**. This project aims to digitalize traditional Indonesian mom-and-pop shops (*warung*), allowing them to manage business and accept orders online.

### 🌟 Core Features:
- **Product Catalog:** Displaying available items and goods beautifully to customers.
- **Stock & Price Management:** Tracking real-time inventory updates and price adjustments.
- **Online Ordering:** Allowing customers to select items and place orders directly through the web.
- **Contact Admin:** Direct chat integration to the store admin via WhatsApp for seamless communication.

## 📚 Roadmap & Curriculum

This learning journey is divided into several milestones, covering everything from version control to frontend, backend, databases, and deployment.

### 1. Version Control System (VCS)
- [ ] **Git & GitHub:** Mastering version control basics (`git init`, `add`, `commit`, `push`, `branching`).

### 2. Frontend Development (Client-Side)
- [ ] **HTML5 & CSS3:** Building semantic layouts and responsive, mobile-friendly designs.
- [ ] **JavaScript (ES6+):** Programming web interactivity and DOM manipulation.
- [ ] **React.js:** Building dynamic, component-based User Interfaces.

### 3. Backend Development (Server-Side)
- [ ] **Runtimes & Languages:**
  - Node.js (JavaScript on the server-side)
  - Python (For alternative backend logic, scripting, or automation)
- [ ] **RESTful API:** Designing secure communication channels between frontend and backend.

### 4. Database (Data Storage)
- [ ] **Database Concepts:** Learning how to securely store products, stock levels, and order data.
- [ ] *Tech to explore:* SQL (PostgreSQL/MySQL) or NoSQL (MongoDB).

### 5. Deployment (Go Live!)
- [ ] **Cloud Hosting:** Deploying the frontend, backend, and database so the *Warung* website can be accessed by anyone online.
